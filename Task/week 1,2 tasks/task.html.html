Web Designing

Web design refers to the design of websites that are displayed on the internet. It usually refers to the user experience aspects of website development rather than software development.

Mobile App Designing:

Mobile app design is the task of designing mobile applications. Even though mobile apps have a wide variety of uses.A mobile app, is a type of application software designed to run on a mobile device, such as a smart phone or tablet computer.

Graphic Designing:

The art or profession of using design elements (such as typography and images) to convey information or create an effect also : a product of this art.

Introduction to World Wide Web:

The World Wide Web (www, W3) is an information space where documents and other web resources are identified by URIs, interlinked by hypertext links, and can be accessed via the Internet. It has become known simply as the Web.

Compare Static websites :

In static web pages, Pages will remain the same until someone changes it manually. Static Web pages are very simple. It is written in languages such as HTML, JavaScript, CSS, etc.

Dynamic websites:

Dynamic web pages are used where the information is changed frequently, for example, stock prices, weather information, etc.
